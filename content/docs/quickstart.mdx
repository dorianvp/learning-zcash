This guide will use [the `zcash-lib` crate](https://github.com/dorianvp/zcash-lib).

Clone the repository:

```bash
git clone git@github.com:dorianvp/learning-zcash.git
```

Install dependencies:

```bash
cd learning-zcash
cargo build
```

And start hacking!
